# TACO Hosted Web App Sample (Visual Studio 2017)

This is the application code that accompanies the Cordova [Hosted Web Application tutorial](NEED A LINK HERE!).

The repository consists of two folders:

+ Starter - Contains the Visual Studio Solution for the web application portion of the project.
+ Completed - Contains the Visual Studio Solution for the web application portion of the project plus the completed Apache Cordova project.
